# Monarch-Revive

Reviving legacy Windows systems with modern security, performance optimization, and lightweight AI tools — built for low-end hardware.

---

## 📌 About (English)

Monarch is a project focused on reviving and enhancing legacy Windows systems (Windows 7 / XP / 8), with a strong focus on Windows 7 SP1.  
It provides security tools, performance optimization, and lightweight AI solutions designed specifically for low-end devices.
